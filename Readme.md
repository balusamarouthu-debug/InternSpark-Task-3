# 📊 Pandas Data Analysis Project

## 📌 Description
This project demonstrates how to load, explore, and analyze a dataset using the Pandas library in Python.

## 🚀 Features
- Load CSV dataset
- Display first 5 rows
- Show dataset structure (info)
- Generate descriptive statistics
- Exception handling

## 🛠️ Technologies Used
- Python 3
- Pandas
